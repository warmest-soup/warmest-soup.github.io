~TECH Index~
Techniques and classes are what define a character's abilities beyond their generic traits. Techniques are gained from a character's Classes. A character gains one Technique for every rank they have in a Tech Class.
All tech classes currently available are detailed below.
~


All Classes colored blue are considered Skills, while those colored purple are considered Insights.
Combatant
A person's fundemental understanding of violence. Their battle I.Q.. Combatants are those comfortable with the use & application of violence, and excel in combat.
*All combat oriented characters benefit greatly from having this class.

Class Bonus: This class' Rank can be added to Grapple checks, and used for advantage when beginning combat under special conditions.
*These advantages will be contested if used against other Combatants.

Exp Bonus: Survive Battles.
1. Jab C:1
Access to level 1 attacks. These attacks deal half standard damage, and require no recovery moments.
2. Guard C:1
Protect yourself by placing an object or limb ahead of your vulnerabilities. Guard differs from block in that it can be focused.
3. Feignt C:1
A false attack that triggers any stances currently active on the target.
4. Fletch C:1
Allows for movement on the same moment as another action.
5. Riposte C:1
An action taken in the recovery moments after another attack action. If the prior attack parries, this technique is triggered, imediately launching another attack.
6. Haymaker C:1
grants access to level 5 attacks, which deal double standard damage, and require 4 recovery moments.
Shield-Bearer
A defensive soldier, used to operating in cooperation with and in support of allies.

Class Bonus: This class' Rank can be used to focus Guard actions with no card cost.

Exp Bonus: Guard using non-weapon objects.
1. Bash C:1
A thrusting attack performed with a shield that, if it lands, destabilizes the target, and causes a speed check, which if failed causes a knockdown. Any single Shield relavant class may be used to apply penalties to the Knockdown check.
2. Lariat C:1
A swinging attack performed with the edge of a shield. This attack cannot be deflected, and may be performed during the recovery moments of another attacks.
3. Ally Guard C:1
Allows a character to share their shield coverage with any object or entity on an adjacent space.
*The guarded ally will be destabilized if they attempt to move through the guard.
4. Shell Poke C:1
Allows for attacks of a lower power level than a shield relavant class rank to occur during guard actions.
5. Full Cover C:1
Brace yourself wholly behind your shield, leaving no entry for frontal attacks.
*If shield coverage is less than 2, use of this skill will leave the acting character prone.
6. Marching Wall P
Allows for movement to occur on the same moment as guard actions.
Defencer
A defensive and parry oriented duelist, that specializes in cunning defensive and counter plays.

Class Bonus: This class' Rank can be used to focus Guard actions with no card cost.

Exp Bonus: Parry while equipped with a defensive off-hand item.
1. Interceptor (Stance) C:1
Automatically parry incoming attacks with an equipped shield.
2. Long Guardd C:1
Hold a shield out away from the body to Double it's coverage.
*The shield used must have a weight value less than the acting character's strength.
3. Shield Counter C:1
When an attack is parried with a weapon, A counter attack can be performed with a shield, this attack will always destabilize the target. If it hits while they're already destabilized, it will destabilize them for an aditional moment.
4. Tilt Glance C:1
When an attack lands on the acting character's guard, the acting character may deflect their weight, moving the attacker to any adjacent space, while facing the same direction.
5. Shield Toss C:1
Throw your shield, if a luck check is passed, the shield will bounce off the target, and towards a second. Each succesive bounce incurs an additional penalty, and targets a random spot. The throw continues until the shield misses or is caught.
6. Tortoise C:1
Shield relavant classes may be added to Grapple Checks, and shields may be used in Dominance to launch full damage strikes.
Guardian
A protecter. One who would trade their ability to inflict harm for a greater ability to defend others.

Class Bonus: This class' Rank can be used to focus Guard actions with no card cost.

Exp Bonus: Use defensive skill actions to protect others.
1. Defensive Posture (Stance) C:1
Automatically guard incoming attacks that are within focus range. A card is consumed if Dex is used to focus instead of a shield relavant class rank.
This stance ends when the acting character is destabilized, knocked down, an action is performed that interrupts the guard, or the acting character is grappled.
2. Repel C:1
A thrust attack performed with a shield that, if it hits, will send the attack backwards up to a number of spaces equal to or less than a shield relavant class rank. If this attack clashes, all incoming damage is reflected back into the source, and the attacker is destabilized.
3. Impact Cushion P
Halves all incoming damage that lands on guards.
4. Bunker (Stance) C:1
Dig in, and resist all destabilizing and knockdown effects.
This stance ends when the acting character performs an action that intterupts their guard, or they are grappled.
5. Shield Charge C:1-6
Allows a character to rush with their guard up any number of spaces equal to a shield relavant class rank in any direction, and end facing any direction. This action occurs in a single moment. If the acting character collides with another, the target is launched a number of spaces equal to the distance the acting character traveled, and is dealt slam damage relative to the distance traveled, and the hardness of the shield used.
The card cost of this skill raises by one for every space traveled.
6. Glance Direct C:1
Finely adjust the angle of your shield to force an attack to glance into a specific direction, redirecting incoming attacks to adjacent spaces, or reflecting certain types of projectiles.
*Reflected projectiles are aimed using shield relavant classes for accuracy.
Shootist
A gunslinger, well versed in the art of gun fighting. Mobile and aggressive.

Class Bonus: When this class is used to target, all penalties from moving are ignored. This applies to both the shooter and the target.

Exp Bonus: Land shots on enemies in combat.
1. Single Action P
Grants the ability to operate two handed firearms with only one as long as their weight is less than the acting character's strength.
A single target penalty for each missing rank in the shootist class is incurred when shooting this way.
2. Warning Shot P
If a gun has not yet been fired in the current encounter, you can Fire a warning shot which will destabilize all other characters in the encounter, with a chance to paralyze them until an attack action is taken by any other character if a Con Check is failed.
3. Quick draw C:1
Allows a character to draw and shoot with a firearm in the same action (With a target penalty if an ability besides Shootist is used for targeting.)
4. Rush Load C:1
Speed through the reloading process, and complete two steps per action. (Lasts until the weapon is loaded, or until canceled by another action.)
*If the final step is completed on a new action, the character may take one free aim.
5. Panic shot C:1
When another character enters their melee range with a firearm drawn, the use of this technique guarantees a hit on a random location, And can be triggered even when the attacker is unnoticed.
*If the attacker is unnoticed, the directing player must chose whether to activate this technique or not before learning more information about their attacker.
6. Hammer Fan C:1
Attack with a repeating firearm a number of times up to the acting character's shootist rank in a single moment. Each attack will land on a random spot.
*Target rolls for individual shots may be focused.
Sharpshooter
A marksman that rarely misses their shot. Class Bonus: When using this class to target, the acting character may stack double this class rank's worth of Aims.

Exp Bonus: Aim at targets beyond a weapon's effective range.
1. Steady Hands C:1
Double the accumulated target advantages from aiming
2. Empty lungs C:1
Used instead of Fire! to simultaneously attack and gain an additional target Advantage.
3. Run Action C:1
Quickly inspect your equipped firearm to determine the chance of malfunction on the next shot.
4. Field Strip C:1
Quickly address your weapon's condition to reduce the chance of malfunction by 1.
5. Range Survey C:1
Gain a persistant target Advantage for the entirety of the battle that is applied to all subsequent shots.
6. Bullet Drop P
Double the effective range of your equipped firearm.
Trickshoter
A sportsman who enjoys putting on a show, or simply entertaining themselves. One who’s mastered the art of challenging, and esoteric shots. Class Bonus: When this class is used to target, it halves all penalties for non-conventional shots.

Exp Bonus: Land shots with more penalties than advantages.
1. Ill Intent C:1
Can Aim while weapon is holstered, and while occupied.
2. Follow-Up C:1
Retain target advantages gained from Aiming for consecutive follow up shots. Each shot, including the first, will lose one advantage.
*The cost of this tech increases by 1 every time it's sequentially.
3. Action Reload C:1
Fire your weapon, and prime the next shot in a single action. This includes the first two steps of the reload process if neccesary.
4. Hip Shot C:1
Halve the target penalties when firing without Aiming.
5. Hat Trick C:1-6
Gain a target penalty on a shot that, if it hits and doesn’t break, will remove a top layer armor.
*Will always suceed in removing hats.
6. Smooth Operator C:1
Remove a single Target Penalty from any shot.
Brawler
A belligerent whirlwind of violence and noise. Class Bonus: Divides incoming damage from blunt sources (Under 3 sharpness) by Class Rank.

Exp Bonus: Attack with non weapon items.
1. Tavern Toss C:1
Pick up an object and throw it in the same Moment, causes the target to be destabilized, and deal impact damage based on the object.
2. Break Shot P
When a weapon or object breaks during the acting character's attack, the damage it inflicts on the target is doubled.
3. Good Time P
Attacking with non-weapon items deals double damage.
4. Door Pin C:1
Large objects and furniture can be used to pin an enemy against a wall or obstacle. The enemy will be unable to escape until a contested Str/Speed check is passed. Once the grapple ends, the target is always left destabilized.
5. Table Breaker C:1
perform throws onto tables or furniture breaking them, and doubling your damage
6. Brute Throw C:1
Can throw the enemy over furniture and obstacles. obstacle must be within the character’s str number of spaces.
*(Can also be used when in Advantage during a grapple.)
Improvisor
An evasive and resourceful fighter that doesn't want any trouble, but always seems to find it. Class Bonus: an improvisor check can be performed when threatened, to dodge a melee attack (Including thrown items) while maintaining position relative to the attacker.

Exp Bonus:Interact with objects during Open Combat. or flee successfully
1. Panic block C:1
If near a liftable object on the same Moment you're threatened, you can abort any current action to guard with that object instead.
2. Kitchen Sink C:1
When the acting character is parried, near a liftable object, this tech may be activated to strike with it. The object attack has no recovery moments, and cannot be used to destabilize the actor.
3. Cloth Catch C:1
A parry attempt using a thick cloth of some kind to catch bladed weapons. As long as this action overlaps an incoming attack, it will never fail.
Once the weapon is caught, the acting character may choose to perform a relavant check to disarm the attacker.
4. Catch Feint C:1
Small items (Size under 2) can be thrown at enemies to cause a Con check, which if failed aborts their current action, and leaves them destabilized.
5. Rollover C:1
Can vault and scramble over, or under obstacles and enemies.
6. Payback C:1
Allows the acting character to perform the most recent tech that was used against them.

Intoxicant
A combatant reliant on the use of substances, to awaken, or subdue aspects of themselves, that allows them to act uninhibited, and more powerfully than when sober.
!!! Intoxicant Class Techniques become available only when a TOXIC effect has been applied!!! Class Bonus: class rank may be used to modify the acting character’s constitution, when ingesting intoxicants, making them more or less effective.

Exp Bonus: Engage in combat under the influence of intoxicants. This exp is multiplied by the number of combat oriented classes with ranks above 3.
1. Beligerance T:1-5
Intoxicant class rank is added to Strength for the next action or round.
*This Tech costs 1 Toxic Sat for every point added, and does not consume an action.
*If the targeted trait would go beyond 6, leftover points are counted as advantages instead.
2. Party Life T:1-5
Intoxicant class rank is added to Charisma for the next action or round.
*This Tech costs 1 Toxic Sat for every point added, and does not consume an action.
*If the targeted trait would go beyond 6, leftover points are counted as advantages instead.
5. Wobbly T:1-5
Intoxicant class rank is added to Speed for the next action or round.
*This Tech costs 1 Toxic Sat for every point added, and does not consume an action.
*If the targeted trait would go beyond 6, leftover points are counted as advantages instead.
3. Touchy T:1-5
Intoxicant class rank is added to Dexterity for the next action or round.
*This Tech costs 1 Toxic Sat for every point added, and does not consume an action.
*If the targeted trait would go beyond 6, leftover points are counted as advantages instead.
4. Feely T:1-5
Intoxicant class rank is added to Grapple Checks for the next action or round.
*This Tech costs 1 Toxic Sat for every point added, and does not consume an action.
*If the targeted trait would go beyond 6, leftover points are counted as advantages instead.
6. Swig P
Can drink freely from a held container, without costing an action. Amount of sat attained is determined by the roll of a single dice. If the result was less than 6, the difference is spilled from the container.
