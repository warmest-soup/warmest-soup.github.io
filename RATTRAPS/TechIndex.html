~TECH Index~
Techniques and classes are what define a character's abilities beyond their generic traits. Techniques are gained from a character's Classes. A character gains one Technique for every rank they have in a Tech Class.
All tech classes currently available are detailed below.
~


All Classes colored blue are considered Skills, while those colored purple are considered Insights.
Combatant
A person's fundemental understanding of violence. Their battle I.Q.. Combatants are those comfortable with the use & application of violence, and excel in combat.
*All combat oriented characters benefit greatly from having this class.

Class Bonus: This class' Rank can be added to Grapple checks, and used for advantage when beginning combat under special conditions.
*These advantages will be contested if used against other Combatants.

Exp Bonus: Survive Battles.
1. Jab C:1
Access to level 1 attacks. These attacks deal half standard damage, and require no recovery moments.
2. Guard C:1
Protect yourself by placing an object or limb ahead of your vulnerabilities. Guard differs from block in that it can be focused.
3. Feignt C:1
A false attack that triggers any stances currently active on the target.
4. Fletch C:1
Allows for movement on the same moment as another action.
5. Riposte C:1
An action taken in the recovery moments after another attack action. If the prior attack parries, this technique is triggered, imediately launching another attack.
6. Haymaker C:1
grants access to level 5 attacks, which deal double standard damage, and require 4 recovery moments.
Shield-Bearer
A defensive soldier, used to operating in cooperation with and in support of allies.

Class Bonus: This class' Rank can be used to focus Guard actions with no card cost.

Exp Bonus: Guard using non-weapon objects.
1. Bash C:1
A thrusting attack performed with a shield that, if it lands, destabilizes the target, and causes a speed check, which if failed causes a knockdown. Any single Shield relavant class may be used to apply penalties to the Knockdown check.
2. Lariat C:1
A swinging attack performed with the edge of a shield. This attack cannot be deflected, and may be performed during the recovery moments of another attacks.
3. Ally Guard C:1
Allows a character to share their shield coverage with any object or entity on an adjacent space.
*The guarded ally will be destabilized if they attempt to move through the guard.
4. Shell Poke C:1
Allows for attacks of a lower power level than a shield relavant class rank to occur during guard actions.
5. Full Cover C:1
Brace yourself wholly behind your shield, leaving no entry for frontal attacks.
*If shield coverage is less than 2, use of this skill will leave the acting character prone.
6. Marching Wall P
Allows for movement to occur on the same moment as guard actions.

Guardian
A protecter. One who would trade their ability to inflict harm for a greater ability to defend others.

Class Bonus: This class' Rank can be used to focus Guard actions with no card cost.

Exp Bonus: Use defensive skill actions to protect others.
1. Defensive Posture (Stance) C:1
Automatically guard incoming attacks that are within focus range. A card is consumed if Dex is used to focus instead of a shield relavant class rank.
This stance ends when the acting character is destabilized, knocked down, an action is performed that interrupts the guard, or the acting character is grappled.
2. Repel C:1
A thrust attack performed with a shield that, if it hits, will send the attack backwards up to a number of spaces equal to or less than a shield relavant class rank. If this attack clashes, all incoming damage is reflected back into the source, and the attacker is destabilized.
3. Impact Cushion P
Halves all incoming damage that lands on guards.
4. Bunker (Stance) C:1
Dig in, and resist all destabilizing and knockdown effects.
This stance ends when the acting character performs an action that intterupts their guard, or they are grappled.
5. Shield Charge C:1-6
Allows a character to rush with their guard up any number of spaces equal to a shield relavant class rank in any direction, and end facing any direction. This action occurs in a single moment. If the acting character collides with another, the target is launched a number of spaces equal to the distance the acting character traveled, and is dealt slam damage relative to the distance traveled, and the hardness of the shield used.
The card cost of this skill raises by one for every space traveled.
6. Glance Direct C:1
Finely adjust the angle of your shield to force an attack to glance into a specific direction, redirecting incoming attacks to adjacent spaces, or reflecting certain types of projectiles.
*Reflected projectiles are aimed using shield relavant classes for accuracy.
